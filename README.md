# Roy G. Beacon

A simple app for playing with iBeacons.

## Background Color

Driven by the `rssi` value of the closest iBeacon

    * 0-20: red
    * 21-30: orange
    * 31-40: yellow
    * 41-50: green
    * 51-60: blue
    * 61-70: purple
    * 71-90: dark gray
